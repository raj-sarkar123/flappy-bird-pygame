# 🐦 Flappy Bird Clone (Pygame)

A fully functional Flappy Bird clone built using **Python** and **Pygame**, featuring smooth physics, procedural obstacle generation, sound integration, and game state management.

## 🚀 Features

- ✅ Smooth bird physics with gravity & rotation
- ✅ Procedural pipe generation
- ✅ Collision detection system
- ✅ Dynamic difficulty scaling
- ✅ Score tracking system
- ✅ High score saved locally
- ✅ Game state management (Menu / Playing / Game Over)
- ✅ Sound effects (Flap, Score, Death)
- ✅ Background music loop
- ✅ Restart system

---

## 🕹 Controls

| Action | Key |
|--------|------|
| Start Game | Any Key |
| Flap | `SPACE` |
| Restart | Any Key (after Game Over) |

---

## 🛠 Tech Stack

- **Python 3**
- **Pygame**
- Object-Oriented Programming
- Delta-time based movement system
