# 🐦 Flappy Bird Clone (Pygame)

A fully functional Flappy Bird clone built using Python and Pygame.

## 🚀 Features
- Smooth bird physics with rotation
- Procedural pipe generation
- Collision detection system
- Score + High Score saving
- Game state management (Menu, Playing, Game Over)
- Sound effects (Flap, Score, Dead)
- Background music
- Difficulty scaling over time

## 🎮 Controls
- SPACE → Flap
- Any Key → Start / Restart

## 🛠 Tech Stack
- Python
- Pygame
